# The Jarvis CLI Assistant

A Python-based command-line interface (CLI) tool that acts as a personal assistant. This project was built to demonstrate mastery of Python fundamentals including Control Flow, Data Structures, File I/O, and JSON handling.

## Features

Task Manager: Add and view tasks (Persistent storage via .txt).

Contact Book: Save and retrieve contacts (Persistent storage via .json).

Calculator: Continuous loop calculator for basic arithmetic.

Error Handling: Robust handling of missing files and invalid inputs.

## Concepts Applied

Modular Programming (Functions)

File Handling (Read/Write text and JSON)

Global Variables & Scope

Input Sanitization

## How to Run

python jarvis.py
